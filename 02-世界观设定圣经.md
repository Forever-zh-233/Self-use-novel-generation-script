# 世界观设定圣经（原创版）

## 项目定位

本文件是原创小说的世界设定。它吸收“修仙 + 江湖 + 庙堂 + 轻喜剧 + 系统成长”的类型快感，但不沿用源文本的专有名词、人物、地点、门派、功法和剧情桥段。

## 世界架构

### 地理

- **晏朝**：主要舞台，外表是太平王朝，底层已开始腐坏。
- **北砚县**：边地小县，主角起点。荒年、税粮、妖祟、江湖客都从这里露头。
- **云京**：晏朝都城，权力中心。
- **青梧书院**：儒修正统之一，收寒门，也收世家子。
- **玄阙山**：修士宗门聚集之地，讲规矩，也讲利益。
- **雾南泽**：南方瘴地，妖族、蛊师、逃民混居。
- **铁门关**：北境军镇，常年与外族和妖潮对峙。

### 历史

- 晏朝立国二百余年，早年靠军功开国，中期靠文治续命。
- 当朝皇帝年老，储位不稳。
- 地方豪强、宗门、书院、军镇互相牵扯。
- 荒年正在扩大，妖祟事件变多，但朝廷仍在粉饰太平。

### 势力

- **朝廷**：六部、巡夜司、地方县衙。
- **巡夜司**：处理妖祟、邪修、诡案的半官方机构。
- **书院**：青梧书院、鸣石书院等。
- **宗门**：玄阙山各派、散修洞府。
- **江湖**：镖局、药铺、捉妖人、黑市。

## 修炼体系

### 境界划分

本世界采用 **12 境了愿修行体系**，完整定义见 `02-修炼境界.md`，本文件不再另立一套，避免冲突。

三层十二境概览：

- **交换层（1-4 境）**：叩门、通脉、凝元、开窍。了愿是"帮人做事换取力量"。
- **理解层（5-8 境）**：化神、归真、明心、通玄。了愿是"理解他人为何有此愿"。
- **超越层（9-12 境）**：听道、御道、齐物、忘我。了愿是"超越愿本身"。

第一卷沈安从未入门起步，靠了愿系统获得第一批资源，逐步踏入叩门、通脉。境界细节、能力表现、升级节奏一律以 `02-修炼境界.md` 为准。

### 修炼资源

- **青钱**：普通货币。
- **灵砂**：修士通用资源。
- **药引**：丹药与医术材料。
- **法器**：有来历、有代价，不是随手捡神器。

### 愿力物理规则

**愿力是什么**：人心中"想要改变现状"的强烈念头凝聚而成的天地能量。不是灵气，不是法力，是比两者更底层的东西——灵气是天地的呼吸，愿力是天地的心跳。

**怎么产生**：任何生灵产生真心愿望时，愿力自然释放。愿望越强烈、越具体、越发自内心，产生的愿力越多。"想活下去"比"想发财"产生的愿力多十倍。虚假的愿望（口是心非、贪念伪装）不产生愿力。

**自然流转**：正常状态下，愿力散入天地，滋养万物——草木生长、风调雨顺、人心安定都依赖愿力的自然循环。这是世界的底层生态。

**收割机制**：巡夜司在各地布设"截愿阵"，像水坝截流一样把百姓日常产生的愿力截取一部分，汇入朝廷的"国运池"。百姓感觉不到被抽取（像慢性失血），但长期被收割的地区会出现"无愿症状"：人变得麻木、不再有期待、草木枯萎、妖祟滋生。

**沈安的特殊性**：愿录让沈安能在了愿的瞬间直接吸收对方释放的愿力，转化为修炼资源。这是上古守护者留下的工具，本意是"修补世界裂缝"，不是给个人升级用的。别的修士做不到这一点——他们只能用灵砂、丹药等间接资源修炼。

**愿力与修炼的关系**：愿力不直接等于修为，但它是最纯净的修炼燃料。用愿力修炼比用灵砂快三倍，且没有杂质。这就是为什么朝廷要收割——国运本质上是用愿力维持的大型护国阵法。

**枯竭后果（无愿之地）**：愿力被过度收割的区域会变成"无愿之地"——草木不生、人心如死水、连妖祟都不愿待（没有怨气可吃）。这是世界在"死"。第5卷的妖潮本质上是妖族被无愿之地逼出家园的求生行为。

**代价与悖论**：
- 了愿不一定是好事——帮人实现愿望可能害了他（想报仇的人报了仇之后更空虚）
- 有些愿是矛盾的——帮A了愿会伤害B，沈安必须选择
- 有些愿是假的——人以为自己想要X，其实需要Y。帮他实现X不产生愿力（因为不是真心愿）
- 愿越大，沈安自己付出的代价越大——不只是"难度"，是"你自己要失去什么"（寿命、感知、记忆片段）

**愿力分类与冲突规则**：
- 真心愿（发自内心的迫切需求）：产生大量愿力，系统正常奖励
- 贪念/虚荣伪装的愿：产生零愿力，帮了白帮，系统不认
- 恶愿（本质是伤害他人）：产生负值，帮了会反噬沈安（扣寿命/受伤/愿债增加）
- 矛盾愿（帮A必伤B）：愿力取决于沈安的选择——选对了（帮真正需要的那个）给奖励，选错了给惩罚
- 假愿（口是心非）：帮表面愿望=零奖励，帮真实需求=大奖励。考验沈安能否看穿人心
- 公愿（群体愿望）：产生巨量愿力但代价成倍——一个村的愿可能要沈安十年寿命
- 死愿（临终遗愿）：愿力极大但是长期承诺，不是一次性了结，需要持续付出

**系统判定规则**：
- 愿录只认"真心"——嘴上说的不算，心里真正想要的才算
- 同一个人的愿不能反复刷（帮完一次，短期内再帮同一人没有奖励）
- 恶愿有伪装——有人会把恶愿包装成正当需求来骗沈安，帮了才发现是陷阱
- 愿力不能储存——了愿产生的愿力必须当场吸收转化，过了就散入天地

### 世界规则

1. 修炼需要时间、资源和代价。
2. 高境修士不能随意屠戮凡人，会被巡夜司和天律反噬。
3. 朝廷允许宗门存在，但要登记山门、税契、弟子名册。
4. 妖祟不是纯怪物，有些是灾荒、怨气和人心养出来的。
5. 善良有成本，救人会惹麻烦，也会结善缘。

## 主角设定

### 沈安

- 穿越者，前世是普通人。
- 当前身份：游方郎中。
- 身体限制：夜里看得清，白日强光下视线发白，不能久战。
- 性格：低调、务实、心软但不圣母。
- 核心特质：能忍、会算账、出手时不拖泥带水。
- 标志物：竹杖、灰布袍、药箱、一头黑驴。

### 黑子

- 沈安养的黑驴。
- 灵性高，懂人话，但不会说人话。
- 叫声是"昂"。
- 搞笑担当，贪吃，记仇，关键时刻靠谱。
- 不喜欢别人摸耳朵。

### 小满

- 后期加入的小女孩，来历与雾南泽有关。
- 爱装大人，嘴硬，怕饿。
- 与黑子互相嫌弃。

## 核心设定

### 金手指：了愿系统

- 沈安帮人完成心愿，心愿越难，奖励越大。
- 系统不直接送无敌，只给修炼资源——药引、功法残页、术法入门。
- 系统面板用 `【】` 标注。

示例：

```text
【愿录：LV2（7/20）】
【本次了愿：帮寡妇找回丢失的鸡】
【奖励：活血散配方（可炼制叩门期辅助丹药）】
```

### 医术

- 沈安懂一些草药、针法和急救。
- 医术不是万能，救错人会带来麻烦。
- 医者仁心可以推动关系线和伏笔线。

## 禁忌事项

1. 不使用源文本的人名、地名、门派名、功法名。
2. 不复刻源文本名场面和桥段。
3. 主角不是无敌的，不能无限升级。
4. 系统收益要克制，不能一章飞升。
5. 善良不能无成本，江湖也不能全员降智。
