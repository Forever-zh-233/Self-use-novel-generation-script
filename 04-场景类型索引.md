# 场景类型索引（写作时按需读取）

本文件只提供可复用的场景规则和原创伪例，不放源文本长摘录。

## 1. 打斗场景

### 伪例

```text
刀风贴着耳边过去。

沈安往后一退。

脚下的青砖裂了一道缝。
```

### 解析

- 不连续堆招式名。
- 先写动作，再写身体反应和环境反馈。
- 短句切断节奏，制造危险。
- 每 2-3 个动作插入一次感官反馈。

### 可复用规则

1. 动作 -> 反应 -> 感官反馈 -> 动作。
2. 不解释招式原理。
3. 战斗结果要快，不拖泥带水。
4. 打赢也要付出一点成本。

## 2. 日常对话

### 伪例

```text
"你又把豆子吃了？"

黑子低头看地。

地上还剩半粒。
```

### 解析

- 口语化，有生活气。
- 对话简洁，不啰嗦。
- 用小动作制造角色感。
- 日常要顺手推进关系或伏笔。

### 可复用规则

1. 对话不超过两三行就换动作。
2. 用细节代替解释。
3. 轻松场景也要有微小目标。
4. 不要写成纯闲聊。

---

## 3. 情绪爆发

### 伪例

```text
沈安没说话。

他把药箱合上。

"人我救。账，回头算。"
```

### 解析

- 情绪通过动作传递。
- 愤怒时反而更安静。
- 强情绪后要有缓冲。

### 可复用规则

1. 不直接写“他很悲伤/愤怒”。
2. 用沉默、停顿、动作暗示。
3. 情绪爆发要推动选择。
4. 不能只爆不变。

---

## 4. 景物描写

### 伪例

```text
雪落了一夜。

北砚县的牌楼白得像没睡醒。
```

### 解析

- 简洁，不堆辞藻。
- 景物服务于情绪或情节。
- 不超过两三句。

### 可复用规则

1. 景物描写不超过三句。
2. 必须带出氛围或剧情信息。
3. 用具体意象，不用抽象美词。

---

## 5. 人物初登场

### 伪例

```text
方绾站在门口。

一身黑衣，腰间挂着巡夜司的铜牌。

县衙里的人忽然都不说话了。
```

### 解析

- 先写身份或动作。
- 再写一两个关键特征。
- 通过旁人反应侧面烘托。

### 可复用规则

1. 不堆外貌。
2. 三句内建立印象。
3. 用行为让角色站住。

---

## 6. 章末钩子

### 伪例

```text
更鼓响到第三声。

门外，有人也敲了三下。
```

### 解析

- 简短，留悬念。
- 用动作、声音、物件、半句话结尾。
- 不解释悬念。

### 可复用规则

1. 章末不要强行升华。
2. 钩子要接本章冲突。
3. 最后一行尽量短。

---

## 7. 系统面板

### 伪例

```text
【守夜经：LV1（120/1000）】
【今夜收益：寿元+3日，听风术熟练度+2】
```

### 解析

- 用 `【】` 标注。
- 面板简洁。
- 面板出现时要改变局面或制造期待。

### 可复用规则

1. 每次面板只给必要信息。
2. 不解释系统长篇机制。
3. 面板后最好接角色反应。

---

## 8. 转场

### 伪例

```text
.........

三日后，北砚县外。
```

### 解析

- 用长省略号分隔场景。
- 直接交代时间地点。
- 不写冗长过渡。

### 可复用规则

1. 转场要干脆。
2. 新场景第一句要有画面或行动。
3. 不滥用“与此同时”。

---

## 9. 群像互动

### 伪例

```text
裴照刚要开口。

黑子已经把供桌上的青果叼走了。

道士脸都绿了。
```

### 解析

- 焦点在主要角色。
- 次要角色用动作和短对白带过。
- 不给每个人平均笔墨。

### 可复用规则

1. 群戏要有主视角。
2. 一次只突出一个笑点或冲突。
3. 人多时少解释，多让反应碰撞。

---

## 10. 内心独白

### 伪例

```text
沈安看着那枚铜钱。

亏了。
```

### 解析

- 内心独白短。
- 不要连续自我剖析。
- 可以用冷幽默表达心态。

### 可复用规则

1. 内心独白不超过一句。
2. 用动作暗示为主。
3. 不直接写“他想”也可以。

---

## 11. 反派压迫

### 伪例

```text
那人把刀放在桌上。

茶碗震了一下。

"选吧。"
```

### 解析

- 威胁来自动作、距离和沉默。
- 对话直接，别解释坏人有多坏。
- 压迫感要落到主角选择上。

### 可复用规则

1. 用物件和动作制造威胁。
2. 少写反派心理。
3. 不让反派长篇自曝。

---

## 12. 喜剧缓冲

### 伪例

```text
黑子打了个响鼻。

然后慢慢把蹄子挪到了钱袋上。
```

### 解析

- 用动物伙伴反应制造笑点。
- 冷幽默，不解释。
- 缓冲不能打断主线太久。

### 可复用规则

1. 笑点要短。
2. 不要让角色集体降智。
3. 喜剧后最好立刻回到当前目标。

---

## 使用说明

写作新章节时：

1. 识别本章主要场景类型。
2. 查阅对应场景规则。
3. 根据本章 beat 选择 1-3 张场景卡。
4. 用场景卡约束写法，不把伪例照抄进正文。
