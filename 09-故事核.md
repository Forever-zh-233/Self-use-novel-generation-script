# 故事核（修订版）

## 项目定位

原创修仙江湖苦中作乐长篇。学习短句节奏、克制叙事、生活感、系统爽点和章末钩子。漂泊结构，每到一地就是一个故事弧。不使用源文本专有内容。

## 一句话故事

一个眼盲心亮的年轻游医，带着一头黑驴走街串巷，帮人了愿换修炼资源，顺手治病、降妖、查冤案，走到哪帮到哪，最后发现最难治的是人心。

**核心叙事模式：治心病。**

沈安是郎中，治的不只是身体的病。每到一个地方，他遇到的"愿"表面上是具体问题（孩子发烧、冤案未雪、妖祟作乱），但底下都是一个人的心结——执念、恐惧、愧疚、不甘。真正的了愿不是解决表面问题，是让那个人放下心里的东西。

这个模式随着沈安的成长而演变：
- 前期（叩门-通脉）：他以为了愿就是"帮人做事"，治的是表面的病
- 中期（凝元-开窍）：他开始看见人心里的结，发现有些愿不是帮了就好的
- 后期（化神-归真）：他能看见心病但不一定能治，学会"有些结只有自己能解"
- 长生之后：见过太多人的心病，从"想帮"到"帮不了"到"接受"到"还是会伸手"——这就是长生的重量

这跟修炼体系完美对应：
- 交换层（1-4境）：了愿是交易，治表面
- 理解层（5-8境）：了愿是理解，看见心病
- 超越层（9-12境）：了愿是超越，不执着于治不治得好

## 主角

**沈安**，游方郎中，穿越者，前世是普通人。

- 表面目标：攒够资源治好眼睛，给自己和黑子找个安稳住处。
- 深层目标：在乱世里保住一点人的体面。
- 身体限制：白天视线模糊，夜里看得清，久战吃亏。
- 内部弱点：心软，容易被"救人"二字拖进麻烦。
- 核心魅力：不喊口号，真遇事会站出来。
- 标志物：竹杖、灰布袍、药箱、一头黑驴。

## 金手指

**了愿系统**（长生型 + 技能解锁型）。

**核心逻辑：你得到了系统，但失去了眼睛。**

眼盲是"入门费"——穿越时就付了，不是每次使用系统才扣。

### 系统运作

- 沈安帮人完成心愿 → 增加寿命 + 解锁能力
- 能力需要自己修炼（有进度百分比）
- 了愿不会让眼睛更瞎（代价已经付过了）
- 系统来源留白——为什么主角有这个系统？结局不揭示

### 系统规则

1. 只认"真心愿"——敷衍的、骗人的、强求的不算
2. 小愿：寿命+1-3年，解锁基础配方/入门功法
3. 中愿：寿命+5-10年，解锁中阶术法/特殊技艺
4. 大愿：寿命+20-50年，解锁高阶功法/秘术
5. 特大愿：寿命+100年以上，解锁顶级术法/逆天能力
6. 愿了账清，不欠不赊
7. 系统面板短，不解释说明书

### 系统面板

基础面板：
```
【姓名：沈安，年龄：22】
【寿命：55，根骨：凡骨】
【状态：失明（左眼模糊，右眼全黑）】
【愿录：LV1（3/10）】
```

了愿结算：
```
【叮咚！愿录LV2（2/20）】
【本次了愿：帮寡妇找回丢失的鸡】
【奖励：寿命+3年，解锁"活血散配方"】
```

### 眼盲设定

- 眼盲是入门费，不是使用费
- 穿越时就付了——得到系统，失去眼睛
- 眼盲本身是永久的，不可逆
- 眼盲带来的"夜视"能力是系统的补偿
- 白天视线模糊，夜里反而看得清
- 后期发展为特殊能力（气息锁定等）

### 等级系统

愿录等级：LV1（1-10次）→ LV2（11-20次）→ LV3（21-40次）→ ...

## 修炼体系

### 境界划分（12境）

详见 `02-修炼境界.md`，核心逻辑：

**初级（了愿是"交换"）：** 叩门→通脉→凝元→开窍
**中级（了愿是"理解"）：** 化神→归真→明心→通玄
**高级（了愿是"超越"）：** 听道→御道→齐物→忘我

### 系统与修炼的关系

- 了愿解锁功法/术法 → 修炼提升 → 突破境界
- 系统给"钥匙"，境界体系给"框架"
- 寿命直接增加，不走境界体系

### 沈安的修炼路线

- 前世是普通人，穿越到一个游方郎中的身体里。原主有扎实的医术底子（针灸、草药、诊脉），沈安继承了这些肌肉记忆和知识。
- **第1章必须交代清楚**：沈安穿越后发现自己会看病，是因为原主的身体记忆，不是他自己学的。这个信息要在第1章通过具体动作（摸到药箱时手自动认出银针/药瓶）自然传达，不要用旁白解释。
- 开局：未入修炼门，靠了愿系统获得第一批资源。
- 修炼方式：药引辅助 + 功法残页 + 实战积累。
- 特殊优势：眼盲带来的"夜视"能力，后期发展为独特术法。
- 修炼节奏：不快，每3-5章一个小突破，每卷一个大境界。

### 了愿系统与修炼的关系

- 了愿不直接给修为，给资源。
- 资源需要自己消化、修炼、融会贯通。
- 大了愿可能给功法残页，需要凑齐才能修炼。
- 少数关键了愿可能触发隐藏奖励或异常反应，并与眼盲之谜相关；普通了愿不强制牵动长线暗线。

## 核心伙伴

**黑子**，黑驴，懂人话，不说人话。负责日常笑点、危险预警、关键时刻补刀。

后期加入：
**小满**，小女孩，来历与雾南泽有关，爱装大人，怕饿。

## 故事结构

**漂泊式**：沈安和黑子从一个地方走到另一个地方，每到一地就是一个故事弧（3-8章）。

每个弧的结构：
1. 到达新地方，遇到有困难的人
2. 了解到对方的心愿
3. 帮忙了愿（中间有波折，可能涉及修炼战斗）
4. 获得系统奖励，修炼资源积累
5. 离开，前往下一站

弧与弧之间有暗线串联：
- 沈安自己的眼盲之谜（为什么眼盲？穿越前是什么人？封印？）
- 某个跨越多地的大事件（妖祸？人祸？天灾？）
- 某个一直在追踪沈安的人/势力

## 主线冲突

明线：沈安一路了愿，攒够治好眼睛的资源和功法。

暗线：他发现自己的眼盲不是病，而是某种封印。只有少数关键了愿或特殊事件会让封印出现微弱反应，并可能引来某些存在的注意；普通系统收益不负责持续泄露真相。

终极问题：治好眼睛的代价是什么？代价他愿不愿意付？

## 核心主题问句（开放，全书不裁决）

这些问题不给标准答案，让事件和选择反复掂量它们。任何角色都不许把它们当道理讲出来。

1. **救一个不想被救的人，是慈悲还是傲慢？**（沈安行医的根本悖论）
2. **人心里的结，到底能不能被外人解开？**（"治心病"这件事本身成不成立）
3. **活得久，是恩赐还是刑罚？**（了愿系统给的长生，对应沈安最深的自欺）

反方立场必须由读者会尊重的角色代言，本卷之内不下结论。

## 主题意象系统（贯穿全书，层层生长）

锁定为"主题意象"(kind=主题意象)，每次复用必须叠加新含义，不许原地复现：

- **盲 / 见**：沈安眼盲却"看得见"人心的结；明眼人睁着眼却看不见身边人的苦。"看见"在本书指向理解与共情，不指向视觉。每次用，问"这次谁真正看见了谁/谁明明睁眼却瞎着"。
- **灯火 / 灰烬**：灯火＝还有人被照看、还有未冷的心愿；灰烬＝熄灭、放弃、被时间磨平的痛。常落在"愿"的收束处。
- （备用，按需启用）**药 / 病**：身体的病与人心的病互为镜像；治得了身、未必治得了心。

archivist 在 motifs_update 里把这几个标 `kind:主题意象`，复用时写 `meaning_add`。

## 第一卷方向

沈安从北砚县出发，一路南行。

第一站：北砚县（起点，了解系统，初入修炼）
第二站：青石镇（行医，遇到第一个真正的妖祟）
第三站：雾南泽边缘（遇到小满，卷入更大的事件）

每站3-8章，第一卷约20章。

## 读者期待

- 看沈安用小本事帮人解决大麻烦。
- 看黑子制造冷幽默和关键反应。
- 看系统奖励一点点积累，修炼逐步突破。
- 看每到一个新地方的新故事、新人物。
- 看苦中有乐，困难总有办法化解。
- 看章末钩子把"这一站"接到"下一站"。
- 看眼盲之谜逐步揭开。

## 禁止方向

1. 主角开局无敌。
2. 一章讲完整个世界观。
3. 系统长篇解释机制。
4. 反派集体降智。
5. 为虐而虐。
6. 复用源文本人物、地名、门派、功法、桥段。
