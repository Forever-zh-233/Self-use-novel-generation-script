# 长线伏笔资产库

本文件管理跨卷、跨数百章的核心伏笔。它不是普通期待账本，不能只写一句话。

使用原则：

1. `08-期待账本.md` 管短中期普通伏笔，通常 5-30 章内需要推进或回收。
2. 本文件管长线伏笔，跨度可以是 100-800 章。
3. 长线伏笔必须有“表层线索、真实含义、禁止提前揭露、内部检查窗口、外显条件、回收条件”。
4. Writer 可以看到写作安全版；Beat Planner、Archivist、连续性检查可以看到完整卡片。
5. 每章不必推进所有长线伏笔。内部检查不等于正文外显，S 级伏笔也允许长时间沉睡。

## 永久钉子

这些内容每章规划前供 Beat Planner 做内部一致性检查。它们是整本书的暗线，不靠模型记忆；Writer 只看安全版，不看真实含义。

- 沈安的眼盲不是普通病，而是某种封印或代价的表象；不要早早解释清楚。
- 了愿系统的来源保持未知，前中期只表现规则和代价，不解释来历。
- 黑子的异常灵性不是普通动物聪明，它能提前感知部分异常；真相后置。
- 北砚夜案只是第一卷入口，背后要能接到更大的“人祸/妖祸/愿债”暗线。

## 外显总规则

- 不要直接提“这是伏笔”，不要在正文出现 `LF-XXX` 编号。
- 不要机械地每隔固定章节提一次；固定的是内部检查，不是正文露出。
- 只有满足“场景自然 + 有新信息增量 + 不破坏神秘感”时，才允许外显。
- 外显可以是误诊、物件反应、角色反应、异常失败、沉默、绕开答案；不一定是正面提示。
- 同一个长线伏笔连续两次外显时，必须换渠道、换语境、换信息增量。
- 一次外显后进入冷却期，除非主线强相关，不要短期反复提。
- 长线伏笔可以整卷不外显，只在卷级整理时确认“仍有效、未遗忘”。

## 长线伏笔卡片

### LF-001 沈安眼盲之谜

- 等级：S
- 生命周期：全书主线，预计后期回收
- 首次埋设：开篇
- 表层线索：白天视线模糊，夜里看得清；对某些异常有特殊感知。
- 真实含义：眼盲与封印/代价有关，和了愿系统来源存在深层联系。
- 读者承诺：主角治眼睛不是单纯升级奖励，而是会牵出身份、代价和选择。
- 禁止提前揭露：禁止在前期直接解释封印来源、幕后存在、系统真相。
- 内部检查窗口：每 20-40 章检查一次是否需要外显；不是强制出现。
- 外显条件：只有出现医治眼疾、夜间异常、误诊、旧物反应、系统副作用等自然场景时才外显。
- 外显方式：视力异常、夜间感知、他人误诊、旧物反应；每次只给一个小增量。
- 下次候选窗口：第5-8章若出现夜间查案，可安排一次“夜里看见别人看不见的细节”的表层外显；若场景不自然则跳过。
- 当前状态：未回收

### LF-002 了愿系统来源

- 等级：S
- 生命周期：全书主线，接近结尾回收或留白
- 首次埋设：第1章
- 表层线索：系统只认真心愿；愿了账清；大奖励常伴随麻烦。
- 真实含义：系统不是单纯外挂，它和愿债、寿数、因果有关。
- 读者承诺：系统每次奖励都在把沈安推向更大的因果账。
- 禁止提前揭露：禁止让系统自述来源，禁止出现说明书式解释。
- 内部检查窗口：每 10-20 章检查一次规则边界；不是每次都写系统异常。
- 外显条件：只有本章发生了愿望、奖励、愿主反悔、假愿、代价转移等事件时才外显。
- 外显方式：假愿不算、真愿有代价、奖励附带副作用、愿主反悔。
- 下次候选窗口：第3-6章若出现小愿奖励，可让奖励带来新麻烦；若没有愿望事件则跳过。
- 当前状态：未回收

### LF-003 黑子的异常灵性

- 等级：A
- 生命周期：前中期持续，后期揭示一部分
- 首次埋设：第1章
- 表层线索：黑子懂人话但不说人话；能提前察觉异常；关键时刻反应不似普通驴。
- 真实含义：黑子和妖祟/愿债/旧案之间有不明关联。
- 读者承诺：黑子不是纯搞笑角色，它的反应能成为破案和保命线索。
- 禁止提前揭露：禁止让黑子说人话，禁止让旁白直接解释它的来历。
- 内部检查窗口：每 5-15 章检查一次黑子是否有自然反应点；不是每次都神秘预警。
- 外显条件：只有遇到异常声音、妖祟气息、旧物、危险靠近、沈安判断失误时才外显。
- 外显方式：拒绝靠近某物、只对特定声音反应、护主、偷吃打断危险。
- 下次候选窗口：第4-5章若巡夜司物件和主线强相关，可安排一次黑子反应的表层外显；若只是普通审问则跳过。
- 当前状态：未回收
