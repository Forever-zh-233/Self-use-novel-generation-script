# 主角能动性与有代价的胜利

读者弃书的头号原因：主角被剧情推着走、靠运气和巧合得救、打谁都顺。修正它，靠下面三条。

## 主角要主动
- 每场戏开始前，先定清楚：沈安**此刻主动想要什么**，为此**做了什么决定**。
- 如果一场戏里他只是被动反应、承受、旁观——没有自己的目的和选择——这场戏就是软的，重写：给他一个想要的东西，让他为它动手。
- 他可以犹豫、可以选错，但必须是"他在选"，不是"事情发生在他身上"。

## 结局只能是"是,但…"或"否,而且…"
- 一场冲突/尝试的结果，禁止写成"是，然后顺利"。
- **是，但是**：他达成了目标，却付出代价 / 带出新麻烦（救了人，暴露了行踪）。
- **否，而且**：他没达成，而且处境更糟（没问出话，反而打草惊蛇）。
- 顺风顺水=没有张力。每次"得到"都要绑一个"失去"或"新问题"。

## 胜利前先让他败一次
- 关键的打脸/逆转/破局之前，先安排一次铺垫性的挫败：他试过、失败了、被逼到角落。
- 读者的爽感来自"挣扎之后的逆转"，不是"一上来就碾压"。
- 失败要真，不是装出来的——他真的差点办砸，代价真的疼。

## 变强必须付代价
- 任何能力提升、境界突破、金手指动用，都要换来**新的代价 / 失去的东西 / 新的限制**。
- 沈安的盲就是天然的代价设定：力量来自别处，眼睛永远换不回。
- 禁止"白嫖式变强"——天上掉好处、毫无代价地越级。
