# 修炼线编织

## 核心规则(每条≤2行,这是写手必读)
- 修炼不独立成段,而是融入日常行动(做饭、赶路、疗伤、战斗恢复),用生活化场景承载
- 突破的契机绑定剧情需要(受伤需要愈合、被追杀需要突破、获得资源才具备条件)
- 变强必伴随新代价/新限制/新敌人,升级是新问题的开始而非终点

## 规则 R-001: 身体锚点反复呈现修为存在感
描述: 不靠旁白介绍修为等级,而是通过身体的具体感知反复锚定修为的存在——突破时的疼痛、运转功法时身体某处的热感或震动、施展能力后的疲劳。这些身体细节在日常场景中也能出现(如运功调息时的微妙变化),让修为始终"在场"。
覆盖: 强
反例: 修为只在正式战斗或修炼章节中被提及,日常段落中完全"消失"。
可复用指令: 在任何涉及角色身体感受的段落中,加入一处与修为相关的小感知(如体内某处微热、手臂隐隐酸胀),不超过一句话。

## 规则 R-002: 瓶颈用"啃"来写
描述: 修炼瓶颈不写为"主角努力修炼但就是突破不了"的重复描写,而是通过一个具体的、反复的、带有挫败感的行为来呈现——反复尝试同一个动作、每天固定时间运功但身体毫无反应、消耗珍贵材料却无效果。瓶颈的"啃"是日常性的,不是高潮性的。
覆盖: 中
反例: 瓶颈只在突破前一刻被提及,缺乏长时间的积累和挫败。
可复用指令: 在日常场景中,让角色做一个与修炼相关的小动作(运功、尝试某技能),以失败或无变化结束,不加感慨,连续出现2-3次后读者自然感受到瓶颈。

## 规则 R-003: 突破绑定剧情危机
描述: 修为突破的契机不是"修炼够了自然突破",而是与剧情中的危机、需求或机缘直接绑定——被逼入绝境时、获得稀有资源时、为保护他人必须变强时。突破本身是剧情推进的一部分,不是独立的"修炼回"。
覆盖: 强
反例: 在安全环境中闭关数月自然突破,与剧情主线脱节。
可复用指令: 将突破的时机安排在角色面临具体生存/战斗需求的时刻,让突破成为解决问题的关键一环。

## 规则 R-004: 修炼与生活融合的幽默
描述: 修炼过程中加入生活化细节和幽默元素——修炼失败后被同伴调侃、修炼时被日常事务打断、用生活化比喻解释功法原理。让修炼不总是严肃的,保持日常质感。
覆盖: 中
反例: 修炼段落全篇严肃、庄重,与日常段落的轻松语感割裂。
可复用指令: 在修炼描写中加入一个小插曲(被干扰、失败后的自嘲、用食物/动物比喻修炼状态),不超过2句。

## 规则 R-005: 升级即新代价
描述: 每次修为提升、获得新能力或新装备,同步引入一个明确的新代价或新限制——副作用、消耗增加、引来觊觎、需要更长恢复时间、旧伤加重。让升级成为推动新剧情(新危机、新抉择)的契机。
覆盖: 强
反例: 角色变强后没有任何新负担,直接碾压下一阶段,失去张力。
可复用指令: 在角色突破或获得新能力的同一段落或下一段,写一个与之相关的代价或副作用,用身体感受或具体后果呈现。

## 规则 R-006: 修为通过他人反应间接展示
描述: 修为的进步不靠主角自评或旁白等级描述,而是通过对手的震惊、盟友的重新评估、旁观者的议论来间接展示。角色自身可能对进步浑然不觉或觉得理所当然。
覆盖: 中
反例: 每次修为提升都用内心独白"我感觉变强了"来确认。
可复用指令: 在角色展示新能力后,不写角色自身感受,转写对手/旁观者的一个微小反应(后退、沉默、重新打量)。
