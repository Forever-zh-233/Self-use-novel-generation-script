## 金手指

**了愿系统**（长生型 + 技能解锁型）。

**核心逻辑：你得到了系统，但失去了眼睛。**

眼盲是"入门费"——穿越时就付了，不是每次使用系统才扣。

### 系统运作

- 沈安帮人完成心愿 → 增加寿命 + 解锁能力
- 能力需要自己修炼（有进度百分比）
- 了愿不会让眼睛更瞎（代价已经付过了）
- 系统来源留白——为什么主角有这个系统？结局不揭示

### 系统规则

1. 只认"真心愿"——敷衍的、骗人的、强求的不算
2. 小愿：寿命+1-3年，解锁基础配方/入门功法
3. 中愿：寿命+5-10年，解锁中阶术法/特殊技艺
4. 大愿：寿命+20-50年，解锁高阶功法/秘术
5. 特大愿：寿命+100年以上，解锁顶级术法/逆天能力
6. 愿了账清，不欠不赊
7. 系统面板短，不解释说明书

### 系统面板

基础面板：
```
【姓名：沈安，年龄：22】
【寿命：55，根骨：凡骨】
【状态：失明（左眼模糊，右眼全黑）】
【愿录：LV1（3/10）】
```

了愿结算：
```
【叮咚！愿录LV2（2/20）】
【本次了愿：帮寡妇找回丢失的鸡】
【奖励：寿命+3年，解锁"活血散配方"】
```

### 眼盲设定

- 眼盲是入门费，不是使用费
- 穿越时就付了——得到系统，失去眼睛
- 眼盲本身是永久的，不可逆
- 眼盲带来的"夜视"能力是系统的补偿
- 白天视线模糊，夜里反而看得清
- 后期发展为特殊能力（气息锁定等）

愿录等级：LV1（1-10次）→ LV2（11-20次）→ LV3（21-40次）→ ...
