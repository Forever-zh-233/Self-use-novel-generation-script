# 节奏控制

## 核心规则(每条≤2行,这是写手必读)
- 紧张段用超短句、单句成段、拟声词独立成段加速;舒缓段用长短交错、生活化口语放慢
- 高强度冲突后必须插入"呼吸单元"(日常场景/吃饭/闲聊),让读者和角色喘气
- 信息密度随场景类型波动:新环境/战斗=高密度,安定日常=低密度聚焦细节

## 规则 R-001: 超短句群加速节奏
描述: 在动作爆发、危机降临、信息揭示的瞬间,将叙述拆碎为多个≤6字的短句、独立拟声词(啪、唰、嗡)或单字段落,每个短句独立成段,模拟镜头快速切换的顿挫感。
覆盖: 强
反例: 在高潮动作段仍用长复合句,会拖慢节奏、消解紧迫感。
可复用指令: 连续3-5个独立短段,每段一句话或一个词,用动词和拟声词驱动,不加修饰语。

## 规则 R-002: 单句成段制造强调与留白
描述: 将关键反应、重大决定、场景转换或情绪定调的句子从段落中抽出,独立成段,形成视觉和节奏上的停顿。用于:角色的心理定格、动作收尾、对话后的沉默、场景切换的"软着陆"。
覆盖: 强
反例: 所有句子都挤在同一段落里,关键信息无法突出。
可复用指令: 感觉某句话"值得停下来想一想"时,把它从段落中抽出,独立成段,前后各空一行。

## 规则 R-003: 紧张-舒缓高频切换
描述: 将紧张的战斗、逃亡、对峙段落与极度舒缓的日常场景(吃饭、钓鱼、闲逛、看风景)进行快速、频繁的切换,制造强烈反差。切换常通过空间转移或时间跳跃实现,不需过度过渡。
覆盖: 强
反例: 长时间只写紧张场景(连续两章高强度),读者会疲劳;或长时间只写日常,读者会失去牵引。
可复用指令: 连续2-3个紧张段落后,切到一个完全松散的日常场景(食物、环境、闲聊),至少持续一个完整段落。

## 规则 R-004: 呼吸单元—事后消化段
描述: 重大冲突、情感爆发或噩耗之后,安排一个完整的"呼吸单元":角色回到安全环境,通过疗伤、进食、沉默、简短对话或日常劳作,自然回顾事件、做出下一步打算。情绪在此沉淀,为下一轮事件蓄力。
覆盖: 强
反例: 大战结束立刻跳到下一个任务,读者情绪无处安放。
可复用指令: 冲突结束后,用一个场景(至少3-5句)描写角色的日常行为——吃饭、整理、发呆、闲聊——在其中夹带一句对事件的回顾或感受。

## 规则 R-005: 信息密度波动
描述: 根据场景类型调节信息密度:探索新环境、揭示世界观时密集交代信息(新名词、新规则、新角色);安定下来后大幅降低密度,聚焦于人物间的一个眼神、一顿饭、一次散步的细节。
覆盖: 强
反例: 信息密度始终均匀(每段差不多的信息量),叙事缺乏呼吸感。
可复用指令: 在高密度信息段后,至少安排一个信息量极低的段落(纯感官描写或纯动作,不含新设定)。

## 规则 R-006: 日常段落中"忽然"引爆
描述: 在一段平稳的日常描写中,用"忽然""这时""正在此时"等转折词引入突发事件或新冲突,打破平静。让日常成为冲突的反衬板,冲突因对比而更有冲击力。
覆盖: 中
反例: 日常段落结束后常规过渡到下一个冲突,缺少突然性的断裂感。
可复用指令: 在日常场景进行到一半时,用一个短句打断(如一声喊、一个异响、一个来人),然后直接切入紧张节奏。

## 规则 R-007: 结尾阶段快速片段式收束
描述: 在故事收尾阶段,使用密集、快速的片段式叙事,每个片段聚焦一个情感支点或一条线索的归宿,通过场景轮换形成"遍历"感,高密度完成多条线的收束。
覆盖: 中
反例: 结尾仍用单一视角长段落慢慢推进,节奏拖沓。
可复用指令: 结尾章用3-5个短场景快速切换,每个场景只写一个核心瞬间(一个动作/一句话/一个画面),场景间不需过渡。
