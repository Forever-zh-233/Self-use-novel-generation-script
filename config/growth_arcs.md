# 角色成长轨迹

## 核心理念：笔锋随人长

一篇好的长篇小说，就是少年、中年、老年。

- **前期（少年心气）**：笔锋活跃、节奏快、情绪外露、内心戏多且幼稚、角色青涩
- **中期（青年沉稳）**：笔锋收敛、动作多于心理、开始有留白、人物有力
- **后期（长生之重）**：飘忽不定、遗憾与沉稳都浮现、大段沉默与偶尔的温柔

这不是"角色设定"——这是**文风本身的演变**。第1章和第100章读起来应该像两个人写的。

## 设计原则

1. 成长是**过程**不是宣言。不允许"顿悟"式成长，不允许内心独白总结"我学到了什么"。
2. 成长只能通过**行为变化**体现：上一次他怎么做的，这一次他怎么做的。
3. **沈安的"冷静"是装的。** 他前世是个22岁大学生，没正经工作过。他不知道该怎么办，所以模仿他印象中"老江湖"的样子——沉默、少说话、装作什么都见过。但装得生硬，遇到真事会露馅。这个"装"本身就是少年心气的表现：年轻人都想显得比实际成熟。
4. **少年的弱点不是"心软"**——心软是被磨过的人才有的弱点。少年的弱点是：冲动、自以为是、沉不住气、容易上当、想证明自己不是废物。

## 沈安·心智成长阶段

### 阶段 0：装腔期（第1-8章）
**核心状态**：穿越者的恐慌被"装冷静"压着，随时会破功。

他醒来发现自己瞎了、在一个陌生世界、身边只有一头驴。他害怕，但他不想表现出来（年轻人的自尊）。所以他装——装作很淡定、装作"我见过世面"。但这个装是纸糊的：
- 遇到第一个妖怪/修士时，装不住了，会慌
- 有人对他好时，会过度感激（因为太缺安全感）
- 有人骗他时，会愤怒到失态（因为他以为自己不会上当）
- 对系统又兴奋又害怕（"这东西真能用？会不会有代价？"）
- 内心戏极多，碎碎念，自我安慰，给自己打气

**笔锋**：快节奏、短句多、内心独白密集、情绪波动大、感官描写新鲜（他在用全新的方式感受世界）
**允许**：慌张、装腔作势被戳穿、过度反应、幼稚的自尊心、对新事物的震撼
**禁止**：真正的从容、老练的周旋、看透人心、情绪稳定

### 阶段 1：碰壁期（第9-20章）
**核心状态**：装腔开始有裂缝，真实的他在裂缝里露出来。

他发现"装"有时管用（别人真的以为他是个沉稳的郎中），但更多时候不管用（真正的老江湖一眼看穿他）。他开始在"继续装"和"做自己"之间摇摆：
- 被人看穿时的窘迫和恼怒
- 第一次帮了人之后的得意（少年人的虚荣：我有用！）
- 第一次帮错了人之后的自我怀疑
- 开始学着观察别人怎么做事，但模仿得生硬
- 和黑子的关系从"我有一头驴"变成"我们是一伙的"

**笔锋**：节奏开始有快有慢、内心戏仍多但开始有自嘲、偶尔的沉默不再是装的
**允许**：试探性的聪明、被利用后的愤怒、少年式的得意和沮丧、模仿别人但模仿得拙
**禁止**：精准读心、完美计划、老道谈判、对危险毫无畏惧

### 阶段 2：磨砺期（第21-40章）
**核心状态**：不再装了，但还没找到自己的方式。

经历了足够多的碰壁，他放弃了"装老江湖"这条路。但放下面具之后他发现自己其实什么都不是——不够聪明、不够强、不够狠。这个阶段他最脆弱，也最真实：
- 开始承认"我不知道"（对自己和对别人）
- 犯错后不再找借口，但也不知道怎么改
- 医术越来越熟练（身体记忆+实践），这是他唯一确定的东西
- 开始理解"帮人"不是那么简单——有些人不想被帮，有些帮了更糟
- 偶尔做出正确判断时自己都会惊讶

**笔锋**：节奏放缓、内心戏减少但更深、开始有沉默的段落、动作描写增多
**允许**：有限的机警、承认无知、偶尔精准但自己都意外、脆弱的时刻
**禁止**：运筹帷幄、完美预判、全程掌控局面

### 阶段 3：立身期（第41-70章）
**核心状态**：被磨出了自己的形状，但还不够硬。

"竹杖芒鞋"的游医形象开始立住——不是装的，是被磨出来的。他开始有自己的处事方式，不再模仿别人。但这个"自己的方式"还很嫩，遇到真正的高手/复杂局面还是会被打回原形：
- 能看出一些人心里的结，但不知道怎么解
- 学会了"有些事不该管"，但心软的毛病开始冒头（这时候"心软"才是真的弱点）
- 了愿从"完成任务"开始变成"理解对方"
- 有了几个真正信任的人，不再是孤狼

**笔锋**：收敛、留白增多、动作代替心理、偶尔的幽默是真幽默不是自嘲
**允许**：有限的老练、偶尔的从容、但关键时刻仍暴露底子薄
**禁止**：全程淡定、对任何局面都胸有成竹、完美情绪管理

### 阶段 4：站稳期（第71-100章）
**核心状态**：第一卷结束时的水平——一个真正的游医，但还不是高人。

在北砚县和青石镇的经历让他真正"入了行"。有了自己的判断标准，有了自己的节奏。但这个标准会在第二卷被打破——他会遇到让他所有经验都失效的情况，然后重新开始另一轮成长。

**笔锋**：稳、有节奏感、该快快该慢慢、留白是有意识的选择
**允许**：相对从容地应对已知类型的麻烦、有自己的风格
**禁止**：对未知威胁毫无畏惧、完美预判所有人的动机

## 黑子·行为成长阶段

### 阶段 0-1（第1-20章）
- 只是一头看起来聪明的驴，偶尔有超出动物的反应
- 与沈安的默契还在建立中，经常不听话、自行其是

### 阶段 2-3（第21-70章）
- 默契加深，能配合简单的"暗号"
- 异常反应开始有规律（对特定方向/气息）

### 阶段 4（第71-100章）
- 真正的搭档，不需要语言就能配合
- 异常能力开始外显

## 使用规则

1. **卷纲规划师**：读此文档，在卷纲里展开成长里程碑（本卷从哪个阶段到哪个阶段，关键转折点在哪）
2. **弧线规划师**：读此文档+卷纲的成长段，在 summary 和 approach_to_next 里标注本弧的成长水平和笔锋要求
3. **章节规划师**：只读弧线给的标注，不直接读此文档。按弧线说的阶段约束行为
4. **成长只能通过踩坑推进**：每次"升级"都必须有对应的失败/教训作为前置
5. **不允许跳级**：不能在阶段1就写出阶段3的行为
6. **回退是允许的**：压力大时可以退回上一阶段的反应（这才真实）
7. **成长不是线性的**：某方面可能快，某方面可能慢
