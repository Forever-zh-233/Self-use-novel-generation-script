# 沈安医术来源设定冲突修复计划

## 结论

故事核与卷纲在“沈安医术来源”上存在明确冲突。正文第1章实际按故事核落地为“原主身体记忆/手记得”，但卷纲仍写“前世中医粗浅了解/家里小诊所耳濡目染”。这会让后续 planner、writer 或 reviewer 误把能力来源改回前世知识。

## 当前证据

- `09-故事核.md:113-114`：
  - 沈安穿越到游方郎中身体里。
  - 原主有扎实医术底子，沈安继承肌肉记忆和知识。
  - 第1章必须交代会看病来自原主身体记忆，不是自己学的。
- `卷纲/10-卷纲.md:18`：
  - 写沈安靠前世中医粗浅了解、家里小诊所耳濡目染，冒充游医。
- `卷纲/10-卷纲.md:22`：
  - 写他靠触诊和前世知识蒙对。
- `输出/文章/第001章.md:31`：
  - 正文实际落地为“他不记得自己学过这些，但手记得。”

## 影响

- 当前正文正典已经选择“身体记忆”，但卷纲仍给下游输入另一套来源。
- 后续如果 reviewer 按卷纲评判，可能把“身体记忆”误判为偏离卷纲。
- writer 可能在新章节中写出“前世小诊所知识”解释医术，回改主角能力根基。

## 修复建议

1. 以正文和故事核为权威：沈安医术来自原主身体记忆和身体里的医术底子。
2. 修改 `卷纲/10-卷纲.md`：
   - 删除或降级“前世中医粗浅了解/小诊所耳濡目染”。
   - 如需保留前世因素，只能写成“普通现代常识/大学生思维”，不能作为医术来源。
3. 在故事核或卷纲加入“能力来源红线”：
   - 不许把医术解释为前世自学。
   - 不许把沈安写成现代医学专家。
4. 增加 check 测试：
   - 根设定中不得同时出现“医术来自身体记忆”和“靠前世中医知识冒充/蒙对”这类冲突表述。
   - 若保留旧设定，必须标注“旧版/废弃”。

## 验收标准

- 故事核、卷纲、正文第1章对医术来源一致。
- planner 输入中只有一套权威能力来源。
- `scripts/run_tests.py check` 能捕捉未来类似根设定冲突。
